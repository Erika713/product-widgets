# Product widgets

## Technical stacks

- Vue 3 + Typescript
- Tailwind css
- Pinia (store)
- Vitest (unit test)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run Unit Tests with Vitest

```
npm run test
```
